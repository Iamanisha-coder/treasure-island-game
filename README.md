# 🏝️ Treasure Island Game

A classic text-based choose-your-own-adventure game built for the web using HTML, CSS, and JavaScript. 

Your mission is simple: navigate through dangerous crossroads, cross a mysterious lake, and choose the right path to find the hidden treasure. One wrong move, and it's game over!

## 🚀 Live Demo

You can play the live version of the game directly in your browser here
[https://github.com/Iamanisha-coder/treasure-island-game]
---

## 🛠️ Tech Stack

* **HTML5:** For the core structure and game layout.
* **CSS3:** For modern styling, animations, and clean layouts.
* **JavaScript (ES6):** For the interactive game logic and dynamic UI updates.

---

## 📖 How to Play

1. Clone or download this repository.
2. Open the `index.html` file in any modern web browser.
3. Read the story prompt and click the choice buttons to decide your fate!

## 📝 Features
* Dynamic state updates using JavaScript (no page refreshes required).
* Completely responsive web design (works great on mobile and desktop).
* Clean, interactive, and user-friendly interface.
